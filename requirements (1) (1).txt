# -----------------------------------------------------------------------------
# STEP 1 -- Install PyTorch FIRST (before PyG).
# Choose the command matching your CUDA version.
#
# CUDA 11.8 (most common on Colab T4):
#   pip install torch==2.2.1 --index-url https://download.pytorch.org/whl/cu118
#
# CUDA 12.1:
#   pip install torch==2.2.1 --index-url https://download.pytorch.org/whl/cu121
#
# CPU only:
#   pip install torch==2.2.1 --index-url https://download.pytorch.org/whl/cpu
#
# Google Colab: torch is pre-installed; skip this step.
# -----------------------------------------------------------------------------

# -----------------------------------------------------------------------------
# STEP 2 -- Install PyTorch Geometric (after PyTorch).
#
#   pip install torch-geometric
#
# Optional sparse ops (faster for large graphs, CUDA 11.8):
#   pip install torch-scatter torch-sparse \
#     -f https://data.pyg.org/whl/torch-2.2.1+cu118.html
# -----------------------------------------------------------------------------

# -----------------------------------------------------------------------------
# STEP 3 -- Install remaining packages (this file):
#   pip install -r requirements.txt
# -----------------------------------------------------------------------------
